Структура проекта:
Класс Calculate реализован для подсчета решений

В качестве рефакторинга можно вынести функции подсчета первой половины решений и второй в отдельную функцию, а так же вывести в отдельную функцию метод подсчета количества элементов статического массива.

Сложность алгоритма n^3



ДОП ЗАДАНИЕ:

При увеличении чисел внутри графа (квадрат, пятиугольник) во внешнем круге точек, будет n - 1 чисел. В таких графах есть зависимость: сумма двух подряд идущих чисел внутренней фигуры будет составлять последовательность чисел по которым
можно составить внешний круг с равными суммами троек.
